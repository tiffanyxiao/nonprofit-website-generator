How to Use the Website Generator

Step 1: By clicking on the Home page you will be directed to our non-profit form. Fill out all the relevant information and click the submit button.

This is the form to fill out


Step 2: After submitting your form scroll down to the “HTML Code Generated” section. You can ctrl+c everything below everything below the “Copy HTML code from here.”

Image of the submit button


Step 3: Congratulation, you’re done! Enjoy your newly generated non-profit website. You can click on the green "click here to preview your website" button to preview the home page of your website.

Image of the preview button

(this button will only appear once you have submitted the form)


Note: You can edit your website components by changing the fields in the form and submitting again.

How to Host Your Newly Generated HTML Code

Check out this helpful guide on how to host your website here. Note that for this guide, you already have all the code you need and you just need a way to host your website so other people can access it.


Option #1: Github Pages (Recommended)

Github pages is free and relatively easy/quick to use. Find out how to host your github page here. Note: A github account is required.

Option #2: Hosting on a website platform

Websites such as wordpress, weebly and Tumblr allow you to include your own html code.

Option #3: Google Sites

Google sites allow for embedded components to be added, although some adjustments may need to be made.

Option #4: Third-party website host

There are third party cloud services and website hosts available.
